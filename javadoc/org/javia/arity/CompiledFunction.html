Class CompiledFunction

Object
  org.javia.arity.Function
      org.javia.arity.CompiledFunction

public class CompiledFunction
extends Function

CompiledFunction is a function that was parsed from a string expression. It is represented as a sequence of bytecodes which are executed in order to evaluate the function.

Thread safety

CompiledFunction instances are thread-safe (don't require external locking), By default the evaluation is globally serialized (it doesn't take advantage of multiple threads).

You can achive parallel evaluation by creating an instance of EvalContext for each thread, and passing the EvalContext as the last parameter of the eval() methods.
