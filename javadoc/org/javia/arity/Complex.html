org.javia.arity
Class Complex

Object
  org.javia.arity.Complex

public class Complex
extends Object

A complex value, composed of the real part (re) and the imaginary part (im). All the methods that return a Complex (such as add(), mul(), etc) modify the object on which they are called and return it (this), in order to avoid new object creation.
